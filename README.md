##Документация

Для работы с этим проектом требуются дополнительные ресурсы:
https://github.com/Pelmeneed/for_biblioteka.git

Главный экран: на главном экране реализованы графики и гистограммы, которые способствуют быстрому пониманию того, сколько книг взято, сколько было выдано за последние три месяца, а также соотношение выданных книг к книгам, которые не были выданы.

Экран с авторами: на экране отображается список всех авторов, представленных в библиотечном фонде. Реализованы функции добавления, редактирования и удаления записей об авторах, а также поиск по имени.

Экран с издательствами: экран предназначен для ведения справочника издательств. Позволяет добавлять новые, редактировать существующие и удалять записи об издательствах.

Экран с типами экземпляров: на экране ведется справочник типов экземпляров (например, учебник, методическое пособие, журнал). Функционал включает добавление, изменение и удаление типов, которые используются для классификации экземпляров.

Экран с группами: экран служит для управления списком учебных групп. Предусмотрены операции по созданию новых групп, их редактированию и удалению. Данные о группах используются для привязки студентов-читателей.

Экран с читателями: на экране с читателями, помимо добавления читателя в систему, необходимо также связать его с группой, если он является студентом. Поле с выбором привязки группы должно появляться только тогда, когда будет выбран тип читателя «Студент».

Экран для выдачи экземпляров: на экране можно произвести выдачу экземпляров читателю. Можно выдать только тому читателю, у которого стоит булевая отметка, что читателю можно производить выдачу экземпляров. Если читателю нельзя произвести выдачу экземпляра, то он не должен появляться в поиске. Читателю, за раз, можно выдать то количество книг, которое ему необходимо, а не по одному экземпляру.

Экран для получения экземпляров от читателей: на данном экране можно произвести возврат книг по их идентификатору. Любой читатель, вне зависимости от булевой отметки, что читателю можно получать экземпляры, может произвести сдачу экземпляров. Сдача экземпляров реализована пакетная, а не за один раз, то есть, можно вернуть за раз несколько экземпляров.
Экран с экземплярами: помимо основной информации, на данном экране также должен быть отображен идентификатор экземпляра.
